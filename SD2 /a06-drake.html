My MYSQL Reference Page

Database Commands

Commands Syntax
CREATE DATABASE (name); creates a database with the name you provide to terminal
SHOW DATABASES; shows all the databases stored on the device
USE (database Name); selects a database to use and stores tables created inside the database
SHOW TABLES; shows the tables stored inside the database
DROP DATABASE IF EXISTS (database name); drops an existing database along with its stored tables and data

Table Column Commands

Commands Syntax
CREATE TABLE (table name); creates a table with the specified name
DROP TABLE (table name); drops a table with the name specified
EXPLAIN (table name); explains the contents of a table
SELECT * FROM (table name); shows specified data from the table

Column Attributes and Modifiers

Commands Syntax
CREATE TABLE (table name); creates a table with the specified name
ALTER TABLE (table name); adds a new column to the table

Table Data Commands

Commands Syntax
INSERT INTO (table name) VALUES (table info); inserts new rows into the table
UPDATE (table name) SET (information you want to change to); updates certain rows in the table
DELETE FROM (table name); deletes certain table data
ALTER TABLE (table name); adds a new column to the table
